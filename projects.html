MarkPy

Tweet #MarkPy
Download on the Mac App Store

MarkPy is a lightweight IDE for Python scripting. If you want to run your Python script, simply press cmd+R or tap on the menu bar Run button. MarkPy support user installed Python packages, such as NumPy, Matplotlib, SciPy. MarkPy doesn't have any packages pre-installed. If you want to use any other Python packages, please consult the package distributor website.
